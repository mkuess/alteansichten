It is a Laravel 11 and Filament 3 project for a historical archive platform.

The first MVP now has the basic data foundation for:

- countries
- states
- districts
- municipalities
- categories
- places

The next phase starts with simple Filament Resources so the admin backend can manage the data step by step.
