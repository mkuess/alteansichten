1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The admin backend is being built step by step with simple Filament Resources.

The region structure is:

- countries
- states
- districts
- municipalities

CountryResource, StateResource, and DistrictResource already exist.
This task only creates the MunicipalityResource.

Replit is only the development environment.
The project must remain standalone and deployable later on my own Linux VM at spaceship.com.

2. Current state

The following tables and models already exist:

- places

Existing Filament Resources:
