1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend first so entries can be maintained properly in Filament.
The public frontend will be handled later.

Media items and media links now exist as backend data structures.
This task creates the first Filament Resource for media management.

For the first MVP, media links should only support:
- place
- municipality

People and organizations are not part of the first MVP yet.

The Claude backend UI mockup is saved as a later design reference.
Do not apply the custom design yet.

Replit is only the development environment.
The project must remain standalone and deployable later on my own Linux VM at spaceship.com.

7. Test steps

If already created by another command, do not duplicate it.

Run:

php artisan test

Expected:
All existing tests pass.

Open:

/admin

Log in as the local admin user.

Check:

- MediaItemResource is visible in the navigation as “Medien”.
- The create page opens.
- Create one test media item manually, for example:

  - title: Testbild Gasthaus zur Linde
  - external_url: https://example.com/testbild.jpg
  - rights_status: needs_review
  - location_status: exact_place

- Save the record.
- Open it for editing.
- Save without changes.

Also check routes:

index, create, and edit routes for the MediaItemResource are registered.
