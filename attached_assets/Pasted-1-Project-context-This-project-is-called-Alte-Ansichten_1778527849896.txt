1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

The backend already supports regions, categories, places, media, media links, and QR code records.

Now the project needs user submissions so external users can later submit historical material for moderation.

Replit is only the development environment.
The project must remain standalone and deployable later on my own Linux VM at spaceship.com.
