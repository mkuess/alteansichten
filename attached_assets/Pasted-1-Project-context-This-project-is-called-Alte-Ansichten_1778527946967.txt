1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

Submissions are part of the MVP as external public contributions that are later reviewed in the backend.
They are not user accounts and not user profiles.

The intended MVP flow is:

A visitor submits material later through a public form.
The submission appears in the backend.
An admin reviews it.
The admin manually creates or updates places, media items, or stories based on the submission.

No automatic publishing is required.
