1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

The backend already supports regions, categories, places, media, media links, QR codes, and submissions.

Now the project needs a simple reporting / rights notice structure.
This is part of the MVP so visitors can later report problems, rights concerns, wrong information, or request corrections.
