1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

QR code records already exist.
QR code records can be created from places.
The public redirect route /qr/{code} exists and increments scan_count.

Now the backend should make the QR redirect URL easy to see and copy.

2. Current state

Laravel 11 is installed and working.
Filament 3 is installed and available at /admin.
Filament assets load correctly.

Existing QR functionality:

- QR records can be created from PlaceResource
- GET /qr/{code} redirects to target_url
- scan_count increments
- last_scanned_at updates

Existing related files:

- app/Models/QrCode.php
- app/Filament/Resources/QrCodeResource.php
- app/Http/Controllers/QrRedirectController.php
- routes/web.php
