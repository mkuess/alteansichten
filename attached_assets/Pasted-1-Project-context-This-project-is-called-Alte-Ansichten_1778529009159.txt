1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build the backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

QR code records already exist.
QR code records can be created from places.
The public redirect route /qr/{code} exists and increments scan_count.
The QrCodeResource now shows a copyable QR redirect URL and an “Open QR URL” action.

Now the backend should generate actual QR image files for existing QR code records.

2. Current state

Laravel 11 is installed and working.
Filament 3 is installed and available at /admin.
Filament assets load correctly.

Existing QR functionality:

- QR records can be created from PlaceResource
- GET /qr/{code} redirects to target_url
- scan_count increments
- last_scanned_at updates
- /admin/qr-codes shows a copyable QR redirect URL
- /admin/qr-codes has an action to open the QR URL

Existing related files:

- app/Models/QrCode.php
- app/Services/QrCodeService.php
- app/Filament/Resources/QrCodeResource.php
- app/Http/Controllers/QrRedirectController.php
- routes/web.php

3. Exact task

Add backend QR image generation for existing QrCode records.

Generate PNG and SVG files and store their paths in the existing qr_codes table fields:

- png_path
- svg_path

4. What Replit should do

- Choose a simple Laravel-compatible QR code package.
- Prefer a stable, commonly used package that can generate PNG and SVG QR codes.
- Install only the required package.
- Add QR image generation to the existing QR code service or create a small dedicated service if cleaner.
- Generate QR images from the full QR redirect URL:
  - url('/qr/' . $qrCode->code)
- Store generated files in Laravel storage, for example:
  - storage/app/public/qr-codes/{code}.png
  - storage/app/public/qr-codes/{code}.svg
- Update the QrCode record:
  - png_path
  - svg_path
- Ensure the public storage link exists if needed:
  - php artisan storage:link
- Add an action to QrCodeResource:

- Show a success notification after generation.
