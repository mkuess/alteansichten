1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

Places are the central content entity.
Media items, QR codes, submissions, and content reports already exist.
Now the Place edit page should become more useful for content maintenance.

2. Current state

Laravel 11 is installed and working.
Filament 3 is installed and available at /admin.
Filament assets load correctly.

Existing backend resources:

- CountryResource
- StateResource
- DistrictResource
- MunicipalityResource
- CategoryResource
- PlaceResource
- MediaItemResource
- QrCodeResource
- SubmissionResource
- ContentReportResource

Existing functionality:

- Places can be managed
- Media items can be managed
- Media links can connect media to places or municipalities
- QR codes can be created from places
- QR redirect and QR image generation exist
- Submissions and content reports exist
- Media upload and preview exist
