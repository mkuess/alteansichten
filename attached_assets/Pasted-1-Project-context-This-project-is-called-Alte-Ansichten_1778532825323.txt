1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 and Filament 3 project for a historical archive platform.

The current priority is to build backend functionality first so entries can be maintained properly in Filament.
Design refinement and public frontend work will be handled later.

Places are the central content entity.
Media items can already be uploaded and managed.
Media links connect media items to places or municipalities.

The Place edit page already shows QR code information, submissions, and content reports.

Now the Place edit page should also show media linked to this place.

Important Replit environment note:
For development in Replit, APP_URL and ASSET_URL must include :8000.
Do not change these environment URL settings in this task.

3. Exact task

Add a simple media links section or relation manager to the PlaceResource edit page so admins can see media linked to the current place.

If simple and safe, allow creating a media link from the Place edit page.
