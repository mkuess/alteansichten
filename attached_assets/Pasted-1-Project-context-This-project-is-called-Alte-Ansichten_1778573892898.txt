1. Project context

This project is called “Alte Ansichten”.
It is a Laravel 11 / Filament 3 project running in Replit.

The Filament backend is currently styled and working.
Media upload still fails.

The confirmed upload root cause is:

Livewire signed upload URLs are generated with APP_URL including :8000.
However, Replit’s proxy sends the request to Laravel without :8000 in the Host header.
Therefore Laravel reconstructs a different URL and Livewire signature validation fails with HTTP 401.

2. Current state

Current settings:

APP_URL includes :8000
ASSET_URL includes :8000

This keeps admin UI/navigation mostly working, but breaks Livewire uploads.

Confirmed upload failure:

- Upload endpoint: /livewire/upload-file
- HTTP status: 401
- Cause: request()->hasValidSignature() fails in Livewire FileUploadController

The required test is now to remove :8000 from APP_URL and ASSET_URL and verify all affected areas.
