5. Important known possibilities

If /livewire/upload-file returns:

- 401: signed URL mismatch. Most likely APP_URL / proxy / port mismatch.
- 419: CSRF/session mismatch.
- 413: upload size limit.
- 422: validation issue in FileUpload configuration.
- 500: Laravel exception, use laravel.log.
- 502: Replit proxy/server issue.

The file is only 95 KB, so do not focus on upload_max_filesize unless the status is clearly 413.
