## 1. Navigation Badge Counts

Add `getNavigationBadge()` to every Resource file in `app/Filament/Resources/` 
so the count of records is shown next to each menu item in the sidebar.

For each Resource, add this method inside the class:

```php
public static function getNavigationBadge(): ?string
{
    return (string) static::getModel()::count();
}
```

Apply this to ALL files in app/Filament/Resources/ that are plain .php files 
(not subdirectory Resources like PlaceResource/Pages/).

## 2. Fix Dashboard Widget Error

The file `resources/views/filament/widgets/top-municipalities-widget.blade.php` 
causes a server error. The widget class is 
`app/Filament/Widgets/TopMunicipalitiesWidget.php` and it uses 
`Municipality::withCount('places')`.

Check if the `places` relationship exists on the Municipality model 
(`app/Models/Municipality.php`). It should have:

```php
public function places(): HasMany
{
    return $this->hasMany(Place::class);
}
```

If the relationship is missing, add it. Then clear the cache:
```bash
php artisan optimize:clear
```
