In the Laravel Filament project in alte-ansichten/, add linked media 
to the municipality slide-in panel on the Karte page.

## 1. Update Karte.php

Open app/Filament/Pages/Karte.php. Find where municipalities are loaded.
Add media items to the municipality query by eager loading media 
through their places:

Load municipalities with their places and each place's media:

->with(['places' => function($q) {
    $q->with(['mediaItems' => function($mq) {
        $mq->where('status', 'approved')
           ->select('id', 'title', 'file_path', 'external_url', 
                    'year', 'type', 'primary_place_id')
           ->limit(12);
    }])
    ->select('id', 'title', 'municipality_id', 
             'latitude', 'longitude');
}])

## 2. Update the municipality JSON data passed to JavaScript

In the view or page class where municipality data is mapped to JSON,
add a media array that collects all media from all places of 
the municipality:

'media' => $m->places->flatMap(fn($p) => $p->mediaItems->map(fn($mi) => [
    'id'        => $mi->id,
    'title'     => $mi->title,
    'file_path' => $mi->file_path,
    'year'      => $mi->year,
    'type'      => $mi->type,
]))->values()->toArray(),

## 3. Update the slide-in panel HTML for municipalities

In resources/views/filament/pages/karte.blade.php, find the 
municipality panel section (where selectedType === 'municipality').

After the existing info (name, postal code, places count, 
Gemeinde bearbeiten link, places list), add a media section:

Show a heading "Medien" only if media array is not empty.

For each medium show a small row with:
- if file_path exists: thumbnail img 40x40px object-cover rounded
- if no file_path: a small icon placeholder
- title (or type if no title)
- year if available

If municipality has no media, show:
"Keine Medien verknüpft."

## 4. Make sure MediaItem has the right relationship on Place

In app/Models/Place.php, verify this relationship exists:

public function mediaItems()
{
    return $this->hasMany(MediaItem::class, 'primary_place_id');
}

If it does not exist, add it.

php artisan optimize:clear
